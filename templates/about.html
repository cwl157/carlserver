CarlServer is an experiment to create a functioning web app entirely on an android phone! (I apologize for any typos in advance.)

The app uses angularjs 1.6.x and all source code is available on github.

All development environments need developer tools and android is no different. Below is the list of tools I use to create and maintain this site. If you know of other, or better ones, hit me up on twitter.

  • DroidEdit(Free with ads)
  • Servers Ultimate Pro(Paid and I think quite expensive but I got it free on an amazon deal)
  • sGit
